## 二、恒星基础科普

### 2.1 什么是恒星？

恒星是由炽热气体组成的球形天体，通过内部核聚变反应产生能量发光。我们的**太阳**就是一颗恒星。

**恒星的三大基本特征**：

| 参数 | 含义 | 单位 |
|------|------|------|
| **温度 (Temperature)** | 恒星表面的有效温度，反映热辐射特征 | K（开尔文） |
| **光度 (Luminosity)** | 恒星辐射的总功率，以太阳光度为单位 | L☉ |
| **半径 (Radius)** | 恒星的大小 | R☉（太阳半径） |

> 💡 **科普小知识**：太阳表面温度约 5800K，而红巨星温度只有约 4500K，但光度却是太阳的数十倍！

### 2.2 恒星的三种类型

本系统科普聚焦于三种最重要的恒星类型：

#### 🌟 主序星（Main Sequence Star）

**定义**：像太阳一样，稳定燃烧氢元素的恒星。

**特征**：
- 温度：约 3,000K - 10,000K
- 光度：0.1 - 1000 L☉
- 处于恒星演化的"壮年期"

**你知道吗？**
> 宇宙中约 **90%** 的恒星都是主序星！包括我们的太阳。

#### 🔴 红巨星（Red Giant）

**定义**：离开主序带、膨胀变大的老年恒星。

**特征**：
- 温度：约 3,000K - 5,000K（相对较低）
- 光度：约 10 - 1000 L☉（非常高）
- 半径：是太阳的 10-100 倍

**你知道吗？**
> 当太阳燃料耗尽时，也会膨胀成红巨星，届时它的半径将延伸到地球轨道！

#### ⚪ 白矮星（White Dwarf）

**定义**：恒星演化的最终产物之一，体积小但密度极高。

**特征**：
- 温度：约 10,000K - 30,000K（表面灼热）
- 光度：约 0.001 - 0.1 L☉（非常暗弱）
- 半径：只有地球大小

**你知道吗？**
> 一勺白矮星物质的重量可达数吨！

### 2.3 什么是赫罗图？

**赫罗图（H-R Diagram）**是天文学家最重要的工具之一，由丹麦天文学家赫茨普龙和美国天文学家罗素分别在 1911 年和 1913 年独立绘制。

**赫罗图的秘密**：

```
                    光度 (L☉)
                      ↑
                      │
           红巨星区域  │        ● 主序星
           (右上角)    │       /
                      │      /
                      │     /  ← 主序带
                      │    /
                      │   /
                      │  /
    白矮星区域 ───────┼─────────────────→
        (左下角)     │   温度 (K) →
                 高温← │ →低温
```

**为什么重要？**

1. **揭示演化规律**：恒星在赫罗图上有规律地分布，不是随机散落
2. **主序带**：90% 的恒星集中在从左上到右下的主序带上
3. **演化轨迹**：恒星一生会在赫罗图上"旅行"，从主序星到红巨星，最终变成白矮星或中子星

## 三、系统功能详解

- 📊 **概率展示**：直观看到该恒星属于各类别的概率

**科普知识点**：

- 理解为什么某些参数组合对应特定类型
- 感受天文学家"观测-分析-分类"的工作流程

### 3.2 科普模式 —— 赫罗图探索

通过交互式赫罗图，直观理解恒星分布规律。

**功能特点**：

- 🌡️ **温度-光度散点图**：每颗恒星在图上是一个点
- 🎨 **分类着色**：主序星、红巨星、白矮星用不同颜色标识
- 📍 **区域标注**：清晰显示各类恒星在图上的位置

**科普知识点**：

- 红巨星为什么在右上角（低温但高光度）？
- 白矮星为什么在左下角（高温但低光度）？

### 3.3 科普模式 —— 颜色分析器

通过颜色指数判断恒星温度，体验天文学家的观测方法。

**功能特点**：

- 🌈 **颜色-温度对应**：蓝色=高温，红色=低温
- 🔬 **天文学家方法**：了解如何通过星光颜色判断温度

**科普知识点**：
- 为什么蓝色恒星温度高，红色恒星温度低？（物理学原理）
- 什么是颜色指数（B-V）？

### 3.4 专业人员模式 —— 恒星分类实验台

面向天文专业人员和爱好者，支持完整的机器学习流程。

**功能特点**：

- 📁 **CSV 数据上传**：使用自己的数据集
- ⚙️ **特征配置**：选择用于分类的特征列
- 🧠 **模型训练**：使用高斯朴素贝叶斯算法训练模型
- 📈 **结果评估**：准确率、精确率、召回率、混淆矩阵

**技术细节**：
- 基于 SDSS DR17 真实数据

- 实验记录持久化存储

## 四、算法科普：高斯朴素贝叶斯

### 4.1 什么是朴素贝叶斯？

**朴素贝叶斯**是一种基于概率的分类算法，其核心是**贝叶斯定理**：

$$P(类别|特征) = \frac{P(特征|类别) \times P(类别)}{P(特征)}$$

通俗解释：
- **P(类别)**：这类恒星在宇宙中有多常见？（先验概率）
- **P(特征|类别)**：如果已知是这类恒星，它的温度/光度等参数通常是多少？（似然）
- **P(类别|特征)**：给定这些观测参数，它最可能是哪类恒星？（后验概率）

### 4.2 为什么适合恒星分类？

**关键假设**：各类恒星的所有物理参数服从**正态分布（高斯分布）**

这在天文学中是有物理依据的：
- 同一类型恒星的核心物理过程相同
- 观测到的参数值在平均值附近波动，符合统计学规律

**优势**：
- 计算简单，适合科普演示
- 不需要大量计算资源
- 结果可解释性强

### 4.3 概率校准

本系统使用**温度校准**技术，避免概率值过于极端：

```typescript
// 温度参数控制概率分布的"锐度"
const temperature = 1.5;
// 较小的温度值让分布更平坦，较大的值让分布更尖锐
const calibratedProb = Math.pow(prob, 1 / temperature);
```

这确保用户看到的是**有意义的概率值**，而不是接近 0% 或 100% 的极端数字。

## 五、数据来源

### 5.1 SDSS 斯隆数字巡天

**Sloan Digital Sky Survey (SDSS)** 是有史以来最成功的天文巡天项目之一：

- **时间跨度**：2000 年至今
- **观测范围**：覆盖超过三分之一的天空
- **数据规模**：数十亿天体记录

**本系统使用**：
- SDSS DR17（第十七次数据发布，2021 年）
- 真实的主序星光谱数据
- 基于真实分布模拟的红巨星和白矮星数据

### 5.2 数据字段说明

| 字段 | 含义 | 科普说明 |
|------|------|----------|
| temperature | 表面温度 (K) | 决定恒星颜色和光谱类型 |
| luminosity | 光度 (L☉) | 恒星辐射的总功率 |
| radius | 半径 (R☉) | 恒星的大小 |
| mass | 质量 (M☉) | 恒星含有多少物质 |
| colorIndex | 颜色指数 (B-V) | 通过滤光片测量颜色差异 |

## 九、常见问题

### Q1：系统中的恒星数据是真实的吗？

**答**：部分是真实的，部分是模拟的。

- **主序星**：使用 SDSS DR17 真实光谱数据
- **红巨星/白矮星**：基于真实物理分布模拟（因为在巡天数据中较为稀少）

模拟数据的分布参数（均值、标准差）均基于天文学观测事实。

### Q2：为什么高斯朴素贝叶斯适合这个任务？

**答**：有两个原因：

1. **物理依据**：同类恒星的核心物理过程相同，其参数服从统计规律
