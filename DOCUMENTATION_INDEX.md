   - 阅读 LOCAL_STARTUP_SUMMARY.md
   - 运行 python3 start_local.py
   - 访问应用

2. **日常开发**
   - 使用 start_local.py 启动
   - 参考 LOCAL_DEVELOPMENT.md 开发
   - 使用 QUICK_REFERENCE.md 查询

3. **提交前**
   - 使用 start.py 进行完整测试
   - 参考 STARTUP_GUIDE.md

4. **遇到问题**
   - 查看 QUICK_REFERENCE.md
   - 查看相应的详细指南
   - 查看日志

### ❌ 避免做法

1. ❌ 不要同时运行两个启动脚本
2. ❌ 不要忽视错误日志
3. ❌ 不要修改 docker-compose.yml 后不重建
4. ❌ 不要在 Docker 中修改代码
